# 8-00 世界の見方 (Ways to See The World) — 導入

> [第8部：世界の見方](./README.md) ・ [全体の目次](../README.md)（共通テンプレート・凡例）

**—** カメラの種類を変えて、見回す・キャラを追う・VR で入り込む、の 3 通りの視点を扱います。

Lite は `ArcRotateCamera`（既出）と `createFreeCamera` を備えます。追従カメラ（`FollowCamera`）と WebXR は無いため、前者は `camera.parent`、後者は現状スコープ外です。
