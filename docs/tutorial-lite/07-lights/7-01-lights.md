# 7-01 ライトを灯す (Light the Night) — ○

> [第7部：光と影](./README.md) ・ [全体の目次](../README.md)（共通テンプレート・凡例）

**目的**：点光源・平行光源・スポットライトを配置して夜のシーンを照らす。

**Lite 対応方針**（v1.8 ソースで確認）：

- `createPointLight`（点光源）/ `createDirectionalLight`（平行光源、既出）/ `createSpotLight`（スポット）がそのまま使えます。
- 色や強度は各ライトのプロパティで設定します。

> 動作確認済みサンプルは準備中です。
